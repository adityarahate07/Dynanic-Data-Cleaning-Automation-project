# 🧹 Dynamic Data Cleaning Automation

## 📌 Description

Dynamic Data Cleaning Automation is a Python GUI application that helps users clean Excel data easily using simple options. It is built using Tkinter for the interface and Pandas for data processing.

---

## 🎯 Features

* Select Excel file (.xlsx, .xls)
* Remove duplicate rows
* Remove blank rows
* Trim spaces from text
* Convert text to Title Case
* Save cleaned file automatically

---

## 🛠️ Technologies Used

* Python
* Tkinter
* Pandas

---

## 🚀 How to Run

### Step 1: Install Libraries

```bash
pip install pandas openpyxl
```

### Step 2: Run program

```bash
python app.py
```

---

## 🖥️ Usage

1. Open the application
2. Select input Excel file
3. Choose cleaning options
4. Select output folder
5. Click **Clean Data**

---

## 📁 Output

The cleaned file will be saved as:

```
Cleaned_filename.xlsx
```
